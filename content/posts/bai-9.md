Vậy là mình đã hoàn thành yêu cầu xây dựng Blog cá nhân bằng **Hugo** và **GitHub Pages**.

## Các bước mình đã thực hiện:
1.  **Cài đặt:** Cài Git và Hugo Extended.
2.  **Khởi tạo:** Dùng lệnh `hugo new site`.
3.  **Theme:** Sử dụng theme PaperMod vì tính tối giản và đẹp mắt.
4.  **Content:** Viết các bài chia sẻ về Java và JS bằng Markdown.
5.  **Deploy:** Đẩy code lên GitHub và bật GitHub Pages.

Link source code của mình: [GitHub Repo](https://github.com/lequoctrong/LeQuocTrong_Blog)

Đồ án này không chỉ giúp mình ôn lại kiến thức mạng máy tính mà còn biết thêm về Static Site Generator. Cảm ơn thầy đã hướng dẫn!
